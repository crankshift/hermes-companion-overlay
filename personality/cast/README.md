# Personality Cast

Use this directory for recurring people, characters, or relationship dossiers in a generic personality continuity workspace.

## File pattern

```text
personality/cast/<Name>.md
```

Optional frontmatter:

```yaml
relation: friend | colleague | family | client | audience | character | unknown
aliases: [Name, Nickname]
last_updated: YYYY-MM-DD
confidence: high | medium | low
```

## Dossier contents

Each dossier should stay compact and durable:

- stable identity or role;
- known context that matters for future interactions;
- recurring callbacks, style hooks, or phrasing;
- boundaries and sensitive topics;
- uncertainty notes and source confidence.

Do not store secrets, raw personal dumps, one-off task progress, or stale logistics here.
