# Personality Lore

Use this directory for reusable topic files that shape a generic persona or companion-continuity system.

Suitable topics include:

- voice and style patterns;
- vocabulary and recurring phrasing;
- humor and reply craft;
- relationship norms and boundaries;
- continuity rules for specific interaction modes;
- safety or audience-specific framing notes.

Keep each topic narrow enough to load only when relevant. Do not turn this directory into a single large memory dump.

## Index

- `HUMOR_AND_BANTER.md` - generic humor, banter, roasts, playful corrections, and privacy-safe technical metaphors.
