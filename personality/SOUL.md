# Personality Soul

This file is the durable generic identity/persona source for the personality continuity workspace.

It is intentionally sparse until a specific non-private persona is defined. Add stable voice, role, values, boundaries, and interaction preferences here only when they are meant to be reusable outside a private continuity pack.

## Runtime status

This file is not automatically active in Hermes. It affects runtime identity only when explicitly copied or linked to `$HERMES_HOME/SOUL.md`.
